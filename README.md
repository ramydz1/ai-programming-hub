# 🚀 AI Programming Hub

**منصة ذكاء اصطناعي متقدمة لحل مشاكل البرمجة والكود**

## 🎯 المميزات

- 🤖 **محرر كود ذكي** - اقتراحات ذكية وإكمال تلقائي
- 💬 **مساعد AI** - حل لمشاكل البرمجة بسهولة
- 🔍 **تحليل الأخطاء** - تحديد وإصلاح الأخطاء تلقائياً
- 🎓 **دروس تفاعلية** - تعلم أفضل الممارسات
- 🌙 **Dark/Light Mode** - واجهة مريحة للعينين
- 🌍 **دعم عربي** - واجهة عربية كاملة

## 🛠️ المكدس التكنولوجي

- **الواجهة الأمامية**: Next.js 14 + React 18 + TypeScript
- **التصميم**: Tailwind CSS + Shadcn/ui
- **الواجهة الخلفية**: Node.js + Express
- **قاعدة البيانات**: MongoDB
- **الذكاء الاصطناعي**: Claude API (Anthropic)
- **المصادقة**: NextAuth.js

## 📦 التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/ramydz1/ai-programming-hub.git
cd ai-programming-hub

# تثبيت الحزم
npm install

# إعدادات البيئة
cp .env.example .env.local

# تشغيل خادم التطوير
npm run dev
```

ثم اذهب إلى `http://localhost:3000`

## 🔑 متغيرات البيئة

```env
MONGODB_URI=your_mongodb_url
CLAUDE_API_KEY=your_claude_api_key
NEXTAUTH_SECRET=your_nextauth_secret
```
